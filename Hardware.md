# Hardware

### Example of hardware

- Controller
    - <b>ESP32</b> / Arduino Uno, Nano, Mega
- Motor Driver
    - <b>HR8833</b> / L298, DRV8835, TB6612FNG
- Motor
    - Micro Motor DC 6V
- Sensors
    - IR Flame Sensor with digital output
    - Ultrasonic Sensor - hc sr04
    - Temperature Sensor - MLX90614
    - Color Sensor - VEML6040
