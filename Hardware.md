# Hardware

### Example of hardware

- Controller
    - ESP32 / Arduino Uno, Nano, Mega
- Motor Driver
    - L298, DRV8835, TB6612FNG, HR8833
- Motor
    - Micro Motor DC 6V
- Sensors
    - IR Flame Sensor 
    - Ultrasonic Sensor - hc sr04
    - Temperature Sensor - MLX90614
    - Color Sensor - VEML6040
